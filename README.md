HOW TO USE THIS PROGRAM:
1. Download One Texture Pack archive, and unpack it.
2. Open One Texture Pack folder, and open replace_textures.bat
3. Add your .png file into the One Texture pack folder
4. Enter .png full file name
5. Enter directory of textures you want to change (leave blank for all

WARNING! I recommend using low-quality .png files for Minecraft to work properly. Max quality you can have without lags is 256x256 pixels.
Also, you should have two to the power of n quality for png for resource pack to work properly. Example: 16x16, 32x32, 64x64, etc.
